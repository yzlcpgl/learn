2、迭代法
雅可比迭代法与高斯赛德尔迭代法
（1）雅可比迭代法的函数文件jacobi.m：
function [y,n]=jacobi(A,b,x0,ep)
D=diag(diag(A))
L=-tril(A,-1);
U=-triu(A,1);
B=D(L+U);
f=D\b;
y=Bx0+f;
n=1;
while norm(y-x0)>=ep
x0=y；
y=Bx0+f;
n=n+1；
end
（2）高斯-赛德尔函数（可能不收敛）
Gauss-Serdel迭代法的函数文件gauseidel.m：
function [y,n]=gauseidel(A,b,x0,ep)
D=diag(diag(A));
L=-tril(A,-1);
U=-triu(A,1);
B=(D-L)\U;
f=(D-L)\b;
y=Bx0+f;
n=1;
while norm(y-x0)>=ep
x0=y;
y=Bx0+f;
n=n+1;
end
